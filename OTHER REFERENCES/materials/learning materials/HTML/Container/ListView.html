What is a ListView?
ListView is a view group that displays a list of scrollable items.
The list items are inserted from the source using an Adapter Class.

Note:
The source of list item can be anything ranging from simple Array to Database queries.
The Adapter acts as a bridge between the listview and the source.Thus the Adapter provides access to the data items by fetching the data from the source.
The Adapter is also responsible for making a View for each item in the data set.
Steps of Implementing ListView to display lists in Android:
  1. Define the array or the data source
    2. 
 String[] days={"Sunday","Monday","Tuesday","Wednesday","Friday","Saturday"};

  2. Define the adapter and provide the layout of a single view to the adapter.
    3. 
 ArrayAdapter adapter = new ArrayAdapter(this,android.R.layout.simple_list_item_1,days);
    Here "android.R.layout.simple_list_item_1" is the default layout file available in android for displaying listview.

  3. Set the adapter to the listview
    4. 
 ListView lv = (ListView) findViewById(R.id.idListView);
 lv.setAdapter(adapter);

  4. Attach an onItemClickListener to define what action should take place on click of the list item
    5. 
 lv.setOnItemClickListener(new AdapterView.OnItemClickListener() {
            @Override
            public void onItemClick(AdapterView adapterView, View view, int i, long l) {

            }
        });
