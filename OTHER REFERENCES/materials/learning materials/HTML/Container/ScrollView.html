ScrollView :-
ScrollView adds a scrollbar to the layout content which is beyond the size assigned to the scrollview container
Note:
  1. ScrollView adds scrolling only in the vertical direction.To enable horizontal scrolling we need to use Horizontal ScrollView.
  2. ScrollView can hold only one direct child. This means that, if you have complex layout with more view controls, you must enclose them inside any another standard layout like LinearLayout, TableLayout ,RelativeLayout etc.
  3. ScrollView should not be used with Listview,Gridview, Recyclerview which have their own vertical scrolling.
How to add scrollview to your app layout?
Add the XML tag of scrollview inside XML view.
Add any layout inside scrollview to which you want to add scrollbars if the content spans more than the height specified for the view.


<?xml version="1.0" encoding="utf-8"?>
<ScrollView xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <LinearLayout
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:orientation="vertical">

        <LinearLayout
            android:layout_width="match_parent"
            android:layout_height="wrap_content"
            android:orientation="horizontal">

            <ImageView
                android:layout_width="100dp"
                android:layout_height="200dp"
                android:src="@drawable/img1" />

            <TextView
                android:layout_width="match_parent"
                android:layout_height="wrap_content"
                android:text="View 1" />

        </LinearLayout>

    </LinearLayout>
</ScrollView>
