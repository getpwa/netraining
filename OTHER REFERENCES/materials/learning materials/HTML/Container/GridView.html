GridView :-
GridView is a ViewGroup that displays items in a two-dimensional, scrollable grid.
The grid items are automatically inserted to the layout using a Adapter. (Similar to what we did in the listview).
Steps of using GridView are:
  1. Add the GridView in the Layout XML file
    2. 
<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:id="@+id/idLayout">

    <GridView
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:numColumns="2"
        android:id="@+id/idGridView"/>
</LinearLayout>
  2. Create the DataSource for the GridView in Activity
    3. 
String [] data={"data1","data2","data3","data4","data5","data6","data7","data8"};
  3. Instantiate the Gridview XML into Java Object
    4. 
GridView gridView = (GridView) findViewById(R.id.idGridView);
  4. Define an Adapter for the GridView and set it to the GridView
    5. 
 ArrayAdapter adapter = new ArrayAdapter(this,android.R.layout.simple_list_item_1,data);
 gridView.setAdapter(adapter);
