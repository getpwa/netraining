SearchView:-
SearchView provides a user interface for the user to enter a search query and submit a request to a search provider.
Steps of using SearchView in Android:
  1. Add <SearchView> element to your xml layout file
    2. 
<SearchView
        android:id="@+id/idsearch"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:iconifiedByDefault="false">

        <requestFocus />
    </SearchView>

  2. Get a reference of this view in Java file
    3. 
SearchView searchView = (SearchView) findViewById(R.id.idsearch);

  3. To execute the query add the method setOnQueryTextListener to the searchview
    4. 
searchView.setOnQueryTextListener(new SearchView.OnQueryTextListener() {
            @Override
            public boolean onQueryTextSubmit(String s) {
                return false;
            }

            @Override
            public boolean onQueryTextChange(String s) {
                return false;
            }
        });
