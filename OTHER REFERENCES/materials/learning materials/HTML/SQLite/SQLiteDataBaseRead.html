Steps to Read Values in SQLite Database:-
  1. In DataBaseHelper class create a method to read data with return type as Cursor.
    2. 
 public Cursor getAllData(){
       
 }

  2. Open Database for reading data in the tables using getWritableDatabase().
    3. 
 public Cursor getAllData(){
 SQLiteDatabase db = this.getWritableDatabase();
 }

  3. Write your query for reading values in the rawQuery() method.
    4. It runs the provided SQL and returns a Cursor over the result set.
    
 Cursor res = db.rawQuery("Select * from " + TABLE_NAME,null);

    Thus the complete method would be:-
    
 public Cursor getAllData(){
  SQLiteDatabase db = this.getWritableDatabase();
  Cursor res = db.rawQuery("Select * from " + TABLE_NAME,null);
  return  res;
  }

  4. Now go to your MainActivity.class
    5. Call the method for reading Data from DataBaseHelper.class
    
 DataBaseHelper myDb;
 myDb.getAllData();

    Since the return type of this method is Cursor.We need to store the data in an Object of class Cursor.
    Thus the statment would be,
    
 Cursor res = myDb.getAllData();

    Now, we can get the result by moving cursor to the next row and appending the result in StringBuffer.
    To get the result from the resultset we pass the parameter ColumnIndex in getString() method.
    Thus,
    
 while (res.moveToNext()){
 stringBuffer.append("Id: "+res.getString(0)+"\n");
 }

This is how we read entries from SQLite DataBase
