Android Notification:-
A notification is a message you can display to the user outside of your application's normal UI.
Notifications are intended to inform users about events in your app.

When the system first issues a notification it first appears as an Icon in Notificaton area.


Details of the Notifications can be seen when the user opens the Notification drawer.

To create a Notification we need to use an object of NotificationCompat.Builder class

Required notification contents:-
The notification object must contain
1. setSmallIcon() :- To set a small Icon
2. setContentTitle() :-To set the content Title
3. setContentText():-To set the content Text
Other paramters are optional.

Notification priority:-
We can assign priority levels to the Notifications by NotificationCompat.Builder.setPriority() by passing one of the NotificationCompat priority constants.

There are five priority levels like as bellow :
1) PRIORITY_MAX =2
2) PRIORITY_HIGH =1
3) PRIORITY_DEFAULT =0
4) PRIORITY_LOW =-1
5) PRIORITY_MIN=-2

Code for Creating Simple Notification:-

//build notification
NotificationCompat.Builder mBuilder =new NotificationCompat.Builder(this);
mBuilder.setSmallIcon(R.drawable.notification_icon)
mBuilder.setContentTitle("Simple notification")
mBuilder.setContentText("This is test of simple notification.");

// Gets an instance of the NotificationManager service
NotificationManager notificationManager =(NotificationManager) getSystemService(Context.NOTIFICATION_SERVICE);

//to post your notification to the notification bar
notificationManager.notify(0 , mBuilder.build());
