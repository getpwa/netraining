Fragments in Java:-
It is recommended to create fragments in Java rather than in XML, since we can change the UI in runtime if we are creating the fragments in Java.

Fragment Manager:
A FragmentManager manages Fragments in Android, specifically it handles transactions between fragments.
A transaction is a way to add, replace, or remove fragments.

Every Activity has its own Fragment Manager accessible through getFragmentManager()
It maintains references to all fragments inside the Activity.
Use findFragmentById() or findFragmentByTag() to get reference to a particular Fragment.

Fragment Transactions:
It is responsible for managing the changes to the UI in terms of adding,removing and replacing fragments.
Steps for managing fragment transaction:
  1. Begin a Transaction
  2. Add, remove and replace the desired fragments
  3. Commit the transaction
Steps for creating Fragments in JAVA:
1. Create a class that extends fragment.

public class MyFragment extends Fragment
2. Create a separate layout file or java objects that contain the appearance

<FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context="demo.arjun.com.fragmentapp.MyFragment"
    android:id="@+id/cdcd12121">

    <!-- TODO: Update blank fragment layout -->
    <TextView
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:text="@string/hello_blank_fragment" />

</FrameLayout>
3. Use the onCreateView method to link its appearance.

public class MyFragment extends Fragment{
public View onCreateView(LayoutInflater inflater, ViewGroup container,
                             Bundle savedInstanceState) {
        // Inflate the layout for this fragment
        return inflater.inflate(R.layout.fragment_my, container, false);
    }
}
4. Attach the fragment inside the Activity through Java

//Create an object of your fragment here it is frag
MyFragment frag = new MyFragment();
FragmentManager manager=getFragmentManager();
FragmentTransaction transaction=manager.beginTransaction();
transaction.add(R.id.my_fragmentlayout_name,frag,"frag_tag");
transaction.commit();
