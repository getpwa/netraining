JSON :
JSON stands for Javascript Object Notation.
It is a lightweight data - interchange format.

It consists of key-value pair where,
Key : It is always a string enclosed in quotation marks.
Value: It can be a string, number, boolean expression, array, or object.

Example1: Value containing a String 

 {
	"Name":"Arjun"
 }

Example2 : Value containing an Array 

 {
	"Authors": ["Arjun", "Ganesh ", "Raj", "Ankit"]
 }

Example2 : Value containing a combination of Integer,String,Array and Objects 

  {
	"rootnode": [{
				"Tutorial": "Android Tutorial",
				"Author": {
				"Name": "Arjun",
				"Surname": "Toshniwal"
				},
				"Rating": 5
			  },
			  {
				"Tutorial": "Angular Tutorial",
				"Author": {
				"Name": "Ankit",
				"Surname": "Toshniwal"
				},
				"Rating": 5
			  }]
	}

Note:
A key value pair follows a specific syntax, with the key followed by a colon followed by the value.
Key/value pairs are comma separated.
