JsonParsing :
All the JSON nodes will start with a square bracket or with a curly bracket.
Json Nodes starting with a square brackets represents JSONArray node.
Json Nodes starting with a curly brackets represents JSONObject node.

Thus, if the nodes starts with square brackets use getJSONArray() method
While, if the nodes starts with curly brackets use getJSONObject() method

Example: 

String JSONDATA = "{
  \"rootnode\":{
    \"key1\":\"value1\",
    \"key2\":\"value2\",
    \"key3\":\"value3\",
    \"key4\":\"value4\",
    \"key5\":\"value5\"
  }
}";

JSONObject reader = new JSONObject(JSONDATA);
JSONObject jObject = reader.getJSONObject("rootnode");
String aJsonString = jObject.getString("key1");

Note:
  • If value is string use - getString("STRINGNAME")
  • If value is boolean use - getBoolean("BOOLEANNAME)
  • If value is integer use - getInt("INTEGERNAME)
  • If value is double use - getDouble("DOUBLENAME)
