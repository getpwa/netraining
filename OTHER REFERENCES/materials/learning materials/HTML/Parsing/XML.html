XML :
XML stands for Extensible Mark-up Language.
XML is a markup language much like HTML that was designed to describe data.
An XML Document consists of elements,each element comprises of a start tag, content and end tag.
An XML document must have exactly one root element (i.e., one tag which encloses the remaining tags).
XML differentiates between capital and non-capital letters.
Every XML Document must follow these conditions:-
  • An XML document always starts with a prolog.

  • Every opening tag has a closing tag.
  • All tags are completely nested.
Example:- 

<?xml version="1.0"?>
<communicate>
    <name>Arjun Toshniwal</name>
    <country>India</country>
    <email>arjun.toshniwal@gmail.com</email>
</communicate>

Note:-
XML tags are not predefined we need to define our own tags.

Example: 

<?xml version="1.0" encoding="UTF-8" ?>
<root>
	<rootnode>
		<Tutorial>Android Tutorial</Tutorial>
		<Author>
			<Name>Arjun</Name>
			<Surname>Toshniwal</Surname>
		</Author>
		<Rating>5</Rating>
	</rootnode>
	<rootnode>
		<Tutorial>Angular Tutorial</Tutorial>
		<Author>
			<Name>Ankit</Name>
			<Surname>Toshniwal</Surname>
		</Author>
		<Rating>5</Rating>
	</rootnode>
</root>
