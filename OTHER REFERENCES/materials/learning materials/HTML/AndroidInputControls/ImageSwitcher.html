ImageSwitcher:-
Android ImageSwitcher is a subclass of Android ViewSwitcher class in the hierarchy.
Android ImageSwitcher switches smoothly between two images and thus provides a way of transitioning from one Image to another through appropriate animations.
Note:-
The views added to an ImageSwitcher must all be ImageViews.
Steps for Implementing ImageSwitcher:-

1. Define the ImageSwitcher in XML.

<ImageSwitcher
   android:id="@+id/IdImageSwitcher"
   android:layout_width="wrap_content"
   android:layout_height="wrap_content"
   android:layout_centerHorizontal="true"
   android:layout_centerVertical="true" >
</ImageSwitcher>


2. Create an instance of ImageSwitcher in Java file and get a reference of its XML Component. 

private ImageSwitcher imageSwitcher;
imageSwitcher = (ImageSwitcher)findViewById(R.id.IdImageSwitcher);


3. Implement the ViewFactory interface and implement unimplemented method that returns an imageView.

imageSwitcher.setImageResource(R.drawable.ic_launcher);
imageSwitcher.setFactory(new ViewFactory() {
   public View makeView() {
      ImageView myView = new ImageView(getApplicationContext());
      return myView;
   }
}

4. Add Animation to the ImageSwitcher

Animation in = AnimationUtils.loadAnimation(this,android.R.anim.slide_in_left);
imageSwitcher.setInAnimation(in);
imageSwitcher.setOutAnimation(out);

Methods:

setImageDrawable(Drawable drawable)
Sets a new drawable(image) on the ImageSwitcher.
setImageResource(int resid)
Sets a new image on the ImageSwitcher with the given resource id.
setImageURI(Uri uri)
Sets a new image on the ImageSwitcher with the given Uri.
ImageSwitcher(Context context, AttributeSet attrs)
Creates a new empty ImageSwitcher for the given context and with the specified set attributes.
