Checkbox:-
Checkboxes allow the user to select one or more options from a set.
Typically, you should present each checkbox option in a vertical list.
How to Create CheckBox?
Create a CheckBox in your Layout.
Since CheckBox allows us to select multiple items we need to register a Click Listener for each one.
For example, here are a couple CheckBox objects :

<CheckBox
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Tea"
        android:id="@+id/checkBox1"
        android:onClick="CheckBoxClicked"/>

    <CheckBox
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Coffee"
        android:id="@+id/checkBox2"
        android:onClick="CheckBoxClicked"/>

    <CheckBox
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="ColdDrink"
        android:id="@+id/checkBox3"
        android:onClick="CheckBoxClicked"/>

Within the Activity that hosts this layout, the following method handles the click event for both checkboxes:

public void CheckBoxClicked(View view) {
    // Is the view now checked?
    boolean checked = ((CheckBox) view).isChecked();

    // Check which checkbox was clicked
    switch(view.getId()) {
        case R.id.checkBox1:
            if (checked)
                // Put some code for checkbox1
            else
                // Put some code for checkbox1
            break;
        case R.id.checkBox2:
            if (checked)
               // Put some code for checkbox2
            else
                // Put some code for checkbox2
            break;
          case R.id.checkBox3:
            if (checked)
               // Put some code for checkbox3
            else
                // Put some code for checkbox3
            break;
    }
}
