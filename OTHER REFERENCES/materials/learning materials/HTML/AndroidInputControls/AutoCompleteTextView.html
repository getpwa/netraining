AutoCompleteTextView:-
An editable text view that shows completion suggestions automatically while the user is typing.
The list of suggestions is displayed in a drop down menu from which the user can choose an item to replace the content of the edit box with.

Android AutoCompleteTextView is the subclass of EditText class.
The list of suggestions is obtained from a data adapter and appears only after a given number of characters defined by the threshold.
Steps of Creating AutoCompleteTextView:-
1. Create AutoCompleteTextView in the xml

<AutoCompleteTextView
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:id="@+id/countries_list"/>

2. Get the reference of AutoCompleteTextView in Java class

 AutoCompleteTextView textView = (AutoCompleteTextView)findViewById(R.id.countries_list);

3. Specify the list of suggestions items to be displayed. You can specify the list items as a string array in java or in strings.xml
  • As a string array in Strings.xml
    • 
//In onCreate Method
String[] countries = getResources().getStringArray(R.array.list_of_countries);
ArrayAdapter<String> adapter = new ArrayAdapter<String>(this,android.R.layout.simple_list_item_1,countries);
textView.setAdapter(adapter);

  • As a String Array in Java
    • 
//In onCreate Method
ArrayAdapter<String> adapter = new ArrayAdapter<String>(this, android.R.layout.simple_dropdown_item_1line, COUNTRIES);
textView.setAdapter(adapter);

//String Array in Java
private static final String[] COUNTRIES = new String[] {
            "Belgium", "France", "Italy", "Germany","Georgia", "Spain","Bangladesh","Belgium","Belarus"
    };
