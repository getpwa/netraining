TextSwitcher:-
A TextSwitcher is useful to animate a label on screen.
Whenever setText(CharSequence) is called, TextSwitcher animates the current text out and animates the new text in.

Android TextSwitcher is a subclass of Android ViewSwitcher class in the hierarchy.
Note:- The views added to an TextSwitcher must all be TextViews.
Steps for Implementing TextSwitcher:-

1. Define the TextSwitcher in XML.

<TextSwitcher
   android:id="@+id/IdTextSwitcher"
   android:layout_width="wrap_content"
   android:layout_height="wrap_content"
   android:layout_centerHorizontal="true"
   android:layout_centerVertical="true" >
</TextSwitcher>

2. Create an instance of TextSwitcher in Java file and get a reference of its XML Component.

private TextSwitcher switcher;
switcher = (TextSwitcher)findViewById(R.id.IdTextSwitcher);

3. Implement the ViewFactory interface and implement unimplemented method that returns an imageView.

switcher = (TextSwitcher) findViewById(R.id.idTextSwitcher);
        switcher.setFactory(new ViewSwitcher.ViewFactory() {
            @Override
            public View makeView() {
                TextView textView = new TextView(getApplicationContext());
                textView.setTextSize(18);
                return textView;
            }
        });

4. Add Animation to the TextSwitcher

Animation in = AnimationUtils.loadAnimation(this,android.R.anim.slide_in_left);
switcher.setInAnimation(in);
switcher.setOutAnimation(out);

Methods:

setCurrentText (CharSequence text)
Sets the text of the text view that is currently showing.
setText (CharSequence text)
Sets the text of the next view and switches to the next view. This can be used to animate the old text out and animate the next text in.
TextSwitcher (Context context, AttributeSet attrs)
Creates a new empty TextSwitcher for the given context and with the specified set attributes.
