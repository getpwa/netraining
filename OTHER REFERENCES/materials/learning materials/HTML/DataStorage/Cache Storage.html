Cache Storage:-
The files stored in internal storage by user has to be explicitly deleted by the user explicitly otherwise it would exist till then.
Cache data stored in memory is not a permanent data because the system automatically deletes it if it feels there is shortage of memory.

Where are the files stored?
Files are stored in the directory data/data/<package-name>/cache folder

Steps to store data in cache:-
  1. Use getCacheDir() to open a File that represents the internal directory where your application should save temporary cache files.
  2. Use the File class to create a file inside that cache folder.
  3. Call openFileOutput(String filename) with the name of the file.
Steps to retrieve data from cache:-
  1. Again get a refernce to the cache directory.
  2. Get the file using File class by specifying its name.
  3. Use FileInputStream to read data from the files.
