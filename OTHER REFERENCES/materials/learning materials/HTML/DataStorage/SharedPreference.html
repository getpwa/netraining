Android SharedPreference:- It is the simplest data storing option available in Android. It stores data in key value pairs.
E.g.
Key Value
Username Arjun
Password Newuser@123
location India

Data is stored in XML file in the directory
data/data/<package-name>/shared-prefs folder
What kind of data can be stored?
Any primitive data: booleans, floats, ints, longs, and strings.
This data will persist across user sessions (even if your application is killed).
How to access them?
To get a SharedPreferences object for your application, use one of two methods:
  1. getSharedPreferences(int mode):
    2. Use this if you need multiple preferences files identified by name, which you specify with the first parameter
  2. getPreferences(String name,int mode):
    3. Use this if you need only one preferences file for your Activity.
    Because this will be the only preferences file for your Activity, you don't supply a name.

Modes:
Modes refer to operating modes.
Use 0 or MODE_PRIVATE for the default operation. MODE_PRIVATE: Only your app can acces the file
MODE_WORLD_READABLE:All apps can read the file
MODE_WORLD_WRITEABLE:All apps can write the file
MODE_MULTI_PROCESS:Multiple process can modify the same shared-prefs file.
MODE_WORLD_READABLE & MODE_WORLD_WRITEABLE are deprecated in current versions.
Uses:
  • Check whether user is using your app
  • Check when your app was last updated
  • Remember user credentials
  • Remember user settings
  • Location caching
How to use shared preferences to store data?
  1. Get a reference to a shared preference object
    1. For a single file, call getPreferences(int mode)
    2. For several file , call getSharedPreferences(String name,int mode)
  2. Call the editor
    3. SharedPreference.Editor editor = sharedpreferences.edit();
  3. Use the editor to add the data with the key
    4. editor.putString("name","arjun");
    editor.putString("password","arjun@123");
  4. Commit Editor changes
    5. editor.commit()
How to use shared preferences to retrieve data?
  1. Get a reference to a shared preference object
    1. For a single file, call getPreferences(int mode)
    2. For several file , call getSharedPreferences(String name,int mode)
  2. Use the key provided earlier to get data
  3. Supply default value if data is not found
    4. String name = sharedPreferences.getString("name","N/A");
    String password = sharedPreferences.getString("password","N/A");
