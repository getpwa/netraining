java:
Contains the Java source code files, separated by package names, including JUnit test code.
All the Activity class are stored inside java folder.
For e.g.The Activity class called MainActivity.java is stored in this directory under the package name you mentioned in the wizard.

MainActivity.java:-
The MainActivity.java class looks like this:

package demo.arjun.com.helloworld;

import android.support.v7.app.AppCompatActivity;
import android.os.Bundle;

public class MainActivity extends AppCompatActivity {

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }
}

Why MainActivty is inheriting from AppCompatActivity?
Every activity should inherit from Activity, directly or indirectly. Here our MainActivity inherits from AppCompatActivity which in itself inherits from FragmentActivity which is inheriting from Activity
Thus our MainActivity is indirectly inherting from Activity.
 public class AppCompatActivity extends FragmentActivity
  ↳public class FragmentActivity extends Activity
Thus basically MainActivity extends Activity
