Styles:- A style resource defines the format and look for a UI.
It can specify properties such as height, padding, font color, font size, background color etc

Note:-
Styles in Android share a similar philosophy to cascading stylesheets in web design i.e they allow you to separate the design from the content.

EXAMPLE:-
XML file for the style (saved in res/values/styles.xml): 

<?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="MyTextStyle" parent="@style/Text">
        <item name="android:textSize">25sp</item>
        <item name="android:textColor">red</item>
    </style>
</resources>

Now, this style can be applied to a TextView (saved in res/layout/):


<EditText
    style="@style/MyTextStyle"
    android:layout_width="fill_parent"
    android:layout_height="wrap_content"
    android:text="Hello, World!" />

Steps of applying styles to an element in Android are:-
  1. The root node of the XML file must be the <resources> tag.
  2. Add a <style> element to the file with a name that uniquely identifies the style
  3. Add an <item> element for each property of that style, with a name that declares the style property and a value to be applied to it. The value for the <item> can be a hex color, font type etc
  4. The parent attribute in the <style> element is optional and specifies the resource ID of another style from which this style should inherit properties.
  5. Assign the style attribute to entries in your layout file via the style=‚Äù@style/any_android_element"
Also Android supports style Inheritance much the same way as CSS in WebApplications.
Thus enabling us to inherit properties from an existing style and then define only the properties that you want to change or add.

Example:-

<?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="MyTextStyle" parent="@style/Text">
        <item name="android:textSize">25sp</item>
        <item name="android:textColor">red</item>
    </style>
</resources>




<?xml version="1.0" encoding="utf-8"?>
<resources>
    <style name="MyTextStyle" parent="@style/Text">
        <item name="android:textSize">25sp</item>
        <item name="android:textColor">red</item>
    </style>
   <style name="MyTextStyle.SecondStyle" 
        <item name="android:textSize">30sp</item>
       <item name="android:textStyle">italic</item>
    </style>
</resources>

Predefined Styles:-
  1. Locate the folder where Android SDK is installed.
  2. Goto platforms > android > data > res > values > styles.xml.
  3. This file contains the code for all of Android’s predefined styles.
