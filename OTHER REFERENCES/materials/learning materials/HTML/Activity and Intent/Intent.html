What is an Intent?
An Intent is a messaging object you can use to request an action from another app component

It’s a way to communicate between Android components (not just activities) to request an action from and by different components.
It’s like a message that Android listens for and react accordingly by identifying and invoking the targeted app’s appropriate component (like an Activity, Service, Content Provider, etc.).The components can be within that same application or some other app.

If multiple apps are capable of responding to the message then Android provides the user with a list of those apps from which a choice can be made.
It is known as Chooser

Uses of Intents:
  • Starting an Activity
  • Starting a service
  • Delivering a broadcast
Types of Intent:-
1. Explicit Intent:
While creating an Intent object when we explicitly specify the target component name(e.g Package name or Class name) directly in the intent, it’s an explicit intent. E.g. 

Intent intent = new Intent(this, LoginActivity.class);
startActivity(intent);
Thus specifying that we want to navigate to LoginActivity.class
2. Implicit Intent:
Implicit Intent doesn't specifiy the component.they must include enough information for the system to determine which of the available components is best to run for that intent. E.g. 

Intent intent=new Intent(Intent.ACTION_VIEW);  
intent.setData(Uri.parse("http://www.javatpoint.com"));  
startActivity(intent);
Thus asking the system to find a suitable component to run this URI.
